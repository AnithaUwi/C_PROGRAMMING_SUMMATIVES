# C Programming Summatives

This repository contains five course projects focused on practical C and systems programming skills.

## Projects

### Project 1 - Smart Traffic Light Controller
- Folder: `project1_smart_traffic_controller`
- Topic: Arduino logic, non-blocking timing with `millis()`, simple hardware I/O
- Main files:
  - `arduino/smart_traffic_controller.ino`
  - `docs/wiring_and_pcb_notes.txt`

### Project 2 - Linux Server Health Monitor
- Folder: `project2_linux_monitor`
- Topic: Bash automation, system metrics, background monitoring, alerts/logging
- Main file:
  - `scripts/health_monitor.sh`

### Project 3 - Course Analyzer
- Folder: `project3_course_analyzer`
- Topic: C structs, dynamic memory, CRUD, sorting/searching, file persistence
- Main file:
  - `src/main.c`

### Project 4 - Data Analysis Toolkit
- Folder: `project4_data_toolkit`
- Topic: Function pointers, callbacks, dataset operations, file load/save
- Main file:
  - `src/main.c`

### Project 5 - Multithreaded Web Scraper
- Folder: `project5_multithreaded_web_scraper`
- Topic: POSIX threads, concurrent URL fetching, file output per thread
- Main file:
  - `src/main.c`

## Build and Run

Each project includes its own `README.txt` with exact build/run steps and expected inputs.

General notes:
- Projects 3, 4, and 5 use GCC on Linux/WSL/Git Bash.
- Project 5 requires `curl` at runtime.
- Project 1 runs in Tinkercad/Arduino environment.
